# REAL_TIME_CHAT_APP

## Real-Time Chat Application

A real-time chat application built using **Node.js, Express.js, Socket.IO, HTML, CSS, and JavaScript**. This project enables users to join chat rooms, communicate instantly, view online users, and receive typing notifications in real time.

---

## рҹ“Ң Project Objective

Develop a real-time communication platform that:

* Allows users to join chat rooms
* Sends and receives messages instantly
* Displays online users
* Shows typing indicators
* Provides a responsive and user-friendly interface

## вҡҷпёҸ Installation & Setup

### Clone Repository

```bash
git clone https://github.com/your-username/REAL_TIME_CHAT_APP.git
```

### Navigate to Backend

```bash
cd backend
```

### Install Dependencies

```bash
npm install
```

### Run Server

```bash

```

Server starts at:

```text
http://localhost:5000
```

### Run Frontend

Open:

```text

```

using Live Server.

## рҹ”Ҙ Application Workflow

### Join Room

1. Enter Username
2. Enter Room Name
3. Click Join Room

### Send Messages

1. Type Message
2. Click Send
3. Message instantly appears for all users in the same room

### Online Users

Users currently connected are displayed in the Online Users section.

### Typing Indicator

Displays:

```text
User is typing...
```

when another user is entering a message.
